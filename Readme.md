# How to run

## Browser version

This is the buggy version

1. `yarn install`
2. `yarn demo`
3. Open `localhost:1234` in your web browser.
4. Right click the PDF -> This Frame -> Open in new tab.
5. Save the PDF
6. Open in Evince or XPDF.


## Node version

This one works

1. `yarn install`
2. `node index-node.js`
3. Open `test.pdf` in Evince.
